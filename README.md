# JavaScript Learning

Tài liệu học JavaScript tiếng Việt — Fundamentals, Functions & Closures, Objects & Prototypes, Async (Event Loop / Promise / async-await), Modules & ES6, các topic nâng cao và TypeScript. Tổng hợp & đào sâu các đề tài phỏng vấn JavaScript.

## Stack

- Next.js 15 + Fumadocs + Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```
